# 🧠 MindLock-App
**Lock into your flow. Unlock your potential.**

![Mindlock Design](<iPhone 16 - 25.png>)

### 📖 Overview
Mindlock is a digital wellbeing application designed to fight distractions and help students and professionals reclaim their focus. In a world of notifications, Mindlock acts as a "mental shield," allowing users to deep-work without interruptions.

### 🎨 UI/UX Design (Figma)
The interface was entirely conceptualized on Figma, focusing on "Calm Tech" principles to reduce user anxiety.
* **Status:** UI/UX Prototype complete.
* **Figma Link:** [(https://www.figma.com/design/F2mIjoRC0ZjtH1h8YMTnjJ/Sans-titre?node-id=0-1&t=OzSggq4itv6gg6vT-1)]

### ✨ Key Features
- **Focus Vault:** A space to "lock" distracting notifications.
- **Mindful UX:** Clean interface designed to reduce cognitive load.
- **Deep Work Mode:** Tools to help maintain concentration.
